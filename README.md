# gameplan-react

A gameplan to create a react app

# Usage

```
npm init gameplan https://github.com/ksmithut/gameplan-react my-project -- [options]
```

# Options

```
Usage: create-gameplan https://github.com/ksmithut/gameplan-react <folder> -- [gameplan-options]

Options:
  --help        Show help                                              [boolean]
  --version     Show version number                                    [boolean]
  --name        The name of the project              [string] [default: "react"]
  --typescript  Make it a typescript project          [boolean] [default: false]
  --yarn        Use yarn instead of npm                [boolean] [default: true]
```
